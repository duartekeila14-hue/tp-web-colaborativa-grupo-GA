Descripción
Un sitio web de la galeria virtual de arte del pais, donde se puede encontrar a los artistas mas destacados del pais con sus obras mas
reconocidas y un rememoracion de la historia de la misma y su autor. Tambien hay un apartado de exhibiciones en donde se publican futuros
eventos de exhibicion orientada a artistas independientes para que puedan hacer muestreos de sus obras.

Distribucion de tareas
Bogado, Daiana: artistas.html y css de esa misma vista
Duarte, Keila: index.html y css del home de la pagina
Schibilbain, Valentina: exhibiciones.html y contactos.html. Css de ambas vistas

¿Que dificultades encontraron o como organizaron el trabajo?
Las dificultades que encontramos fueron varias: Recordar los comandos del GitHub y recordar que realizaba cada una de una de ellas. Tambien el correcto posicionamiento en la rama que le correspondia a cada una de los integrantes sin sobreescribir o afectar la rama principal.
Una de las dificultades finales fue la organizacion del CSS para realizar el merge y que no se arme un conflicto entre los codigos.
